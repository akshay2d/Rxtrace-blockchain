## üìû Where to Get GitHub Copilot Help

### For Billing & Subscription Issues

**Your Question**: "I paid $10, why is my bill showing $0? What's the billing cycle?"

**Answer**: 
- GitHub Copilot Individual plan costs **$10/month** or **$100/year**
- If your bill shows $0, you may be:
  - In a free trial period
  - Using a plan included with another subscription (GitHub Enterprise, etc.)
  - Experiencing a billing system issue

**How to Check**:
1. Go to: https://github.com/settings/billing
2. Check your **Billing & Plans** section
3. Look for **GitHub Copilot** subscription status
4. View billing history and next payment date

**If something looks wrong**:
- Contact GitHub Support: https://support.github.com/
- Select "Billing" as your issue type
- Provide your account details

### For GitHub Copilot Not Working

**If Copilot isn't working in VS Code**:

1. **Check your subscription**:
   - Visit: https://github.com/settings/copilot
   - Ensure subscription is active

2. **Verify VS Code setup**:
   - Install/update the GitHub Copilot extension
   - Sign in to GitHub in VS Code
   - Check extension settings

3. **Troubleshoot**:
   - Reload VS Code window
   - Sign out and sign back in
   - Check VS Code output panel for errors

4. **Get help**:
   - VS Code docs: https://code.visualstudio.com/docs/editor/github-copilot
   - GitHub Copilot docs: https://docs.github.com/en/copilot

## üí≥ Understanding GitHub Copilot Billing

### Individual Plan
- **Price**: $10/month or $100/year (prices as of Dec 2024 - check GitHub for current pricing)
- **Billing Cycle**: Monthly on the date you subscribed
- **Free Trial**: 30 days for eligible users (eligibility varies)

> **Note**: Pricing may change. Always check https://github.com/features/copilot#pricing for current rates.

### When You're Charged
- **First charge**: After free trial ends (if applicable)
- **Recurring**: Every month on your subscription date
- **Annual**: Once per year if you chose annual billing

### Why Bill Shows $0
- You're in the free trial period
- Payment hasn't been processed yet
- You have a sponsored/enterprise account
- There's a billing issue (contact support)

### How to Update Payment
1. Go to https://github.com/settings/billing
2. Click **Payment information**
3. Update your payment method
4. Save changes
